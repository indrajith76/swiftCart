## 1) What is the difference between null and undefined?
| Null | Undifined |
| --- | --- |
| null একটি ডাটা টাইপ | undefined মানে যেটি ডিফাইন করা হয় নি। |
| এটি মেমেরিতে জায়গা নেয় বা স্টোর করা যায় | এটিকে একপ্রকার ইরর বলা যায়, যখন ভেরিয়েবলের কোন মান দেয়া থাকে না |

## 3) What is the difference between == and ===?
| == | === |
| --- | --- |
| এটি শুধু মাত্র দুইটি ভেলুকে কম্পেয়ার করার জন্য ব্যবহার হয় | এটি শুধু মাত্র দুইটি ভেলু এবং তাদের টাইপ কম্পেয়ার করার জন্য ব্যবহার হয় |
| Example: ``` (5 == "5") = true ``` | Example: ``` (5 === "5") = false ``` |

## 4) What is the significance of async/await in fetching API data?
যখন আমরা ডাটা ফেচ করি তখন আমাদের ডাটা গুলো API থেকে আসার জন্য ওয়েট করতে হয়, কিন্তু যদি আমরা বাকি পরবর্তী কোড গুলোকে ওয়েট না করাই তখন ইরর আসতে পারে। যদি আমরা async/await ব্যবহার করে API এর ডাটা ফেচ করি তাইলে ডাটা লোড শেষ হলে পরবর্তী কোডের কাজ গুলো হবে।

## 5) Explain the concept of Scope in JavaScript (Global, Function, Block).
জাভাস্ক্রিপ্ট এবং অন্যান্য প্রোগ্রামিং ল্যাংঙ্গুয়েজ গুলোতেও এই Scope আছে। Scope মূলত একটা এরিয়া যেখানে ওই এরিয়ার বাইরে থেকে কেউ ডাটা বা ভেরিয়েবলের ডাটা এক্সেস করতে পারেনা।
* Global: এই Scope এর সকল ভেরিয়েবল এক্সেস করার যাবে অন্য সকল জায়গা থেকে, যেখানে ভেরিয়েবলটা একদম সেই ফাইলে একদম উপরে ডিক্লেয়ার করা হয়ে থাকে। 
* Function: ফাংশনাল Scope মানে একটা ফাংশনের যতটুকু এরিয়া সেকেন্ড ব্রেকেট দিয়ে সেট করা হয় সেটাই ওই ফাংশনের Scope, এর বাইরে থেকে ওই ফাংশনের ডাটা এক্সেস করা যাবে না।
* Block: যেকোন কন্ডিশন বা লুপের মধ্যকার যে ভেরিয়েবলের ডাটা বাইরে থেকে এক্সেস করা যায় না তাকে ব্লক Scope বলে।
